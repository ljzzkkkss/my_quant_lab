## 📖 项目简介

**极客量化实验室** 是一个功能完整的 A 股量化交易回测系统，提供从单股手动回测、参数自动寻优、批量扫描到多股组合轮动的全维度策略分析能力。系统内置 5 大经典量化策略，支持大盘择时、MACD 动能、量比、RSI 等多重过滤条件，帮助量化交易者快速验证策略有效性并找到最优参数组合。

---

## ✨ 核心特性

### 🎯 四大功能模块

| 模块 | 功能描述 |
|------|----------|
| 📊 **手动回测看板** | 单股深度回测，可调参数即时验证，生成详细交易明细与明日实战指令 |
| 🤖 **机器参数寻优** | 暴力扫描参数空间，热力图展示收益分布，支持 OOS 样本外盲测验证 |
| 📡 **雷达全场扫描** | 批量扫描全市场股票，快速筛选策略适配标的 |
| 🧺 **组合轮动** | 多股动态分仓轮动，复利再投资，专业净值曲线与回撤分析 |

### 🧠 五大策略模型

| 策略 | 核心逻辑 | 适用场景 | 风险点 |
|------|----------|----------|--------|
| 📈 **双均线动能** | 短均线上穿长均线买入，MACD 水上过滤 | 捕捉大牛股主升浪 | 震荡市反复打脸 |
| 🌋 **布林带突破** | 收盘价站上中轨看多，跌破看空 | 专抓妖股起爆点 | 假突破较多 |
| 🧲 **RSI 极值反转** | RSI 低于下轨抄底，高于上轨逃顶 | 震荡市印钞机 | 单边暴跌易腰斩 |
| 🌊 **MACD 趋势** | DIF/DEA 金叉买入，死叉卖出 | 波段稳健抗骗线 | 信号滞后 |
| ⚡ **KDJ 震荡** | KDJ 超卖买入，超买卖出 | 短线极其敏锐 | 遇主升浪易踏空 |

### 🛡️ 风控体系

- **全局硬性止盈止损**：可配置固定比例止盈/止损，触发强制平仓
- **大盘择时过滤**：基于沪深 300(510300) 位置决定开仓时机
- **MACD 动能过滤**：金叉时 MACD 必须在水上才确认买入
- **量比过滤**：成交量需达到设定倍数，避免假突破
- **RSI 超买拦截**：RSI 过高时拒绝开仓
- **趋势斜率过滤**：均线斜率需向上，避免下跌趋势接飞刀

## 🚀 快速开始

### 环境要求

- Python 3.8+
- Windows / Linux / macOS

### 安装步骤

1. **克隆项目**
```bash
git clone <your-repo-url>
cd my_quant_lab
```

2. **创建虚拟环境** (推荐)
```bash
python -m venv venv
# Windows
venv\Scripts\activate
# Linux/macOS
source venv/bin/activate
```

3. **安装依赖**
```bash
pip install -r requirements.txt
```

4. **启动应用**
```bash
streamlit run main.py
```

5. **访问界面**

浏览器自动打开 `http://localhost:8501` 即可使用

## 📋 使用说明

### 1. 全局设置 (侧边栏)

- **核心策略模型**：选择 5 大策略之一
- **初始资金**：设置回测起始资金 (默认 10 万)
- **开始/结束日期**：设置回测时间范围
- **全局硬性止盈/止损**：设置固定比例止盈止损 (默认 15%/-8%)
- **过滤引擎**：开启 MACD、大盘择时、量比、RSI、斜率等过滤条件

### 2. 手动回测看板

1. 选择股票 (默认贵州茅台 600519)
2. 调整策略参数 (如短期/长期均线)
3. 设置买入仓位
4. 点击「执行完整回测」
5. 查看明日实战指令、收益指标、交互式 K 线与净值曲线

### 3. 机器参数寻优

1. 设置参数搜索范围与步长
2. 开启 OOS 盲测 (可选)
3. 调整训练集/测试集比例
4. 点击「开启暴力扫描」
5. 查看热力图与前 10 名参数排行榜

### 4. 组合轮动

1. 多选股票构建轮动池
2. 设置最大持仓槽位 (默认 5 只)
3. 开启动态复利 (可选)
4. 查看组合净值曲线、动态回撤饼图、交易流水

## 🔧 核心亮点

### 真实交易模拟

- **100 股整数倍限制**：买入/卖出严格遵循 A 股 100 股最小单位
- **手续费精确计算**：买入万三、卖出万八 (含印花税)，最低 5 元
- **前复权数据**：自动处理分红送配，保证回测准确性

### 增量数据缓存

- **智能台账机制**：首次拉取建立本地 CSV 缓存
- **增量更新**：仅下载缺失区间，节省时间与带宽
- **时间旅行防护**：自动拦截未收盘数据，防止未来函数

### 样本外验证 (OOS)

- **数据切分**：支持自定义训练集/测试集比例
- **盲测排行榜**：前 10 名参数在测试集重新验证
- **状态标注**：✅通过 / ❌崩盘 直观展示过拟合风险

## 📊 数据源

系统使用 **Baostock (证券宝)** 作为数据源，提供 A 股历史行情数据：

- 日线数据 (前复权)
- 自动适配 ETF 降级处理 (不支持复权时自动切换不复权)
- 沪深 300 指数用于大盘择时过滤

> 注意：Baostock 需要联网使用，请确保网络畅通

## ⚠️ 风险提示

1. **历史回测不代表未来收益**：量化策略存在失效风险
2. **参数过拟合**：暴力寻优可能找到历史巧合参数，请务必使用 OOS 验证
3. **交易成本**：已包含佣金与印花税，但未考虑滑点与冲击成本
4. **停牌风险**：回测假设每日可成交，实际可能遭遇停牌无法买卖
5. **本系统仅供学习研究**：不构成任何投资建议，实盘请谨慎
